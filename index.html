⌨️ Keyboard Control:

1 = ปากกาแดง | 2 = ปากกาเขียว | 3 = ปากกาน้ำเงิน

↑ / ↓ = ขยาย/ย่อ Sprite (Scale)

← / → = หมุน Sprite (Rotate)

W / S / A / D = เลื่อนตำแหน่ง (Translate)
